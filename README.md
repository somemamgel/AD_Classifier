# AD Classifier
Alzheimer's disease classifier based on PyTorch and Flask

# Usage
Skull Strip
```bash
python brain_segment.py
```
Train model:
```bash
python train.py
```
Run prediction web application:
```bash
python api.py
```

# Dependence
- Python    3.7.9
- PyTorch   1.8.0+cu111
- Flask     1.1.2
- CUDA      11.0
- cuDNN     8.0.4
- [deepbrain](http://github.com/iitzco/deepbrain)
